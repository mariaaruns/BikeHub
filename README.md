🚲 BikeHub (hobby project)

BikeHub is a .NET-based bike store and service management system designed for internal business use.
It helps manage bikes, customers, inventory, service orders, and staff operations in a single platform.


---

✨ Features

🚴 Bike product management (Brand, Category, Models)

🧾 Order & service tracking

👥 Customer management

🏬 Store & inventory (stock) management

👨‍🔧 Staff management

📦 Order items & service history

🔐 Secure authentication & authorization

📊 Internal reporting and dashboards



---

🛠️ Tech Stack

Backend: .NET 9 Web API(minimal api using carter)

Frontend: .NET MAUI (TESTED ON ANDROID)

Database: SQL Server

MICRO ORM: DAPPER

Authentication: ASP.NET Identity / Custom Auth

Tools & Libraries:

Refit (API communication)

ClosedXML (Excel reports)

Swagger (API documentation)




---

📂 Database Design

Main entities used in BikeHub:

Product

Category

Brand

Stocks

Stores

Staff

Customer

Orders

OrderItems


Designed for internal operations, not a public e-commerce platform.
